# Conquête des Territoires

## Description

Conquête des Territoires is a turn-based strategy game developed in JavaScript. The objective is to control the largest portion of an 8×8 map by deploying units, capturing territories, and defeating enemy forces.

Players must carefully manage their units, expand their territory, and use strategic combat to dominate the battlefield.

## Features

* 8×8 game board
* Two-player gameplay
* Territory capture system
* Multiple unit types with unique abilities
* Turn-based movement and actions
* Dice-based combat mechanics
* Special tiles including bonuses and traps
* Victory through territorial domination or elimination of enemy units

## Unit Types

| Unit     | Movement  | Strength | Special Ability |
| -------- | --------- | -------- | --------------- |
| Soldier  | 1 tile    | Medium   | Versatile       |
| Cavalier | 1–2 tiles | Low      | Fast movement   |
| Tank     | 1 tile    | High     | Strong defense  |

## Game Rules

1. Players place their units in their designated starting zones.
2. Each turn consists of:

   * Dice roll
   * Movement phase
   * Action phase
3. Units can move, attack, defend, or capture territories.
4. Combat is resolved using a dice roll combined with unit strength.
5. The player who controls 33 territories or eliminates all enemy units wins the game.
